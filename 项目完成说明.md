## 🚀 快速开始（5 分钟）

### 第一步：进入部署目录

```bash
cd my-profile
```

### 第二步：选择一种方法配置

**方法 A：使用自动化脚本（推荐）**

**Windows 用户：**
```powershell
# 1. 编辑 update-profile.ps1，在"配置区域"填写你的信息
# 2. 运行脚本
.\update-profile.ps1
```

**Linux/Mac 用户：**
```bash
# 1. 编辑 update-profile.sh，在"配置区域"填写你的信息
# 2. 添加执行权限并运行
chmod +x update-profile.sh
./update-profile.sh
```

**方法 B：手动替换**
1. 用文本编辑器打开 `README.md`
2. 使用查找替换功能（Ctrl+H 或 Cmd+H）
3. 参考 `个人信息配置.txt` 依次替换占位符

### 第三步：部署到 GitHub

1. 登录 GitHub，创建新仓库
2. ⚠️ **仓库名称必须与你的 GitHub 用户名完全相同**
3. 设置为 Public，勾选 "Add a README file"
4. 编辑 README.md，复制粘贴你修改好的内容
5. 提交更改

### 第四步：查看效果

访问 `https://github.com/你的用户名` 即可看到精美的个人主页！

## ⚠️ 重要注意事项

### 部署时必须注意

1. **仓库名称** = GitHub 用户名（完全相同，包括大小写）
2. **仓库设置** = Public（公开）
3. **文件名称** = README.md（注意大小写）
4. **占位符替换** = 所有 "你的GitHub用户名" 必须全部替换
5. **等待刷新** = 首次部署后等待 1-2 分钟

### 常见错误

❌ 仓库名与用户名不一致 → README 不显示  
❌ 忘记替换用户名 → 统计图不显示  
❌ 仓库设为 Private → README 不公开  
❌ 文件名拼写错误 → 无法识别
