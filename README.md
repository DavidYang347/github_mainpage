# GitHub 个人主页模板集合 🎨

这个项目提供了多种风格的 GitHub Profile README 模板，帮助你快速创建一个精美的 GitHub 个人主页。

## 📚 如何创建 GitHub Profile README

### 步骤 1: 创建特殊仓库
1. 登录你的 GitHub 账户
2. 创建一个新仓库，**仓库名称必须与你的 GitHub 用户名完全相同**
   - 例如：如果你的用户名是 `zhangsan`，仓库名就是 `zhangsan`
3. 勾选 "Add a README file"
4. 点击 "Create repository"

### 步骤 2: 编辑 README.md
1. 从本项目的 `templates/` 目录中选择一个你喜欢的风格
2. 复制模板内容到你的仓库 README.md
3. 根据模板中的注释修改为你自己的信息
4. 提交更改

### 步骤 3: 查看效果
访问你的 GitHub 个人主页 `https://github.com/你的用户名`，就能看到精美的 README 了！

## 🎨 可用的风格模板

### 1. **简约专业风格** (`style-1-minimalist.md`)
- 适合：专业开发者、求职者
- 特点：简洁大方、重点突出、易于阅读
- 包含：个人简介、技术栈、GitHub 统计、联系方式

### 2. **动态炫酷风格** (`style-2-dynamic.md`)
- 适合：年轻开发者、开源爱好者
- 特点：动态打字效果、动画徽章、视觉冲击力强
- 包含：动态标题、技能图标、GitHub 动态图表、社交链接

### 3. **极简黑白风格** (`style-3-minimal-bw.md`)
- 适合：追求极简主义的开发者
- 特点：黑白配色、极简设计、突出内容
- 包含：简短介绍、关键技能、项目链接

### 4. **创意卡片风格** (`style-4-creative.md`)
- 适合：前端开发者、设计师
- 特点：卡片式布局、丰富的视觉元素、个性化
- 包含：欢迎横幅、技能卡片、项目展示、贡献图表

### 5. **数据驱动风格** (`style-5-data-driven.md`)
- 适合：数据科学家、后端开发者
- 特点：重数据展示、丰富的统计图表
- 包含：详细的 GitHub 统计、语言使用占比、贡献热力图

## 📁 项目结构

```
github_mainpage/
├── README.md                      # 项目说明（本文件）
├── templates/                     # 模板目录
│   ├── style-1-minimalist.md     # 简约专业风格
│   ├── style-2-dynamic.md        # 动态炫酷风格
│   ├── style-3-minimal-bw.md     # 极简黑白风格
│   ├── style-4-creative.md       # 创意卡片风格
│   └── style-5-data-driven.md    # 数据驱动风格
├── examples/                      # 示例目录
│   └── preview.md                # 各风格预览说明
└── resources/                     # 资源目录
    └── useful-tools.md           # 有用的工具和资源
```

## 🛠️ 自定义建议

1. **个人信息**：修改姓名、职位、简介等基本信息
2. **技术栈**：根据你实际掌握的技术调整技能图标
3. **社交链接**：添加你的 LinkedIn、Twitter、个人网站等
4. **GitHub 统计**：将用户名替换为你自己的
5. **项目展示**：pin 你最自豪的项目

## 💡 小贴士

- 定期更新你的 README，保持信息的时效性
- 添加一些个性化元素，让你的主页与众不同
- 不要过度堆砌元素，保持页面简洁易读
- 确保所有链接都是有效的
- 可以添加一些有趣的元素，比如你最近在听的音乐、正在玩的游戏等
